# LinkedIn DM Guidelines

## Structure
- Max 100 words (LinkedIn messages should be shorter than email)
- No subject line needed
- Conversational tone — this is a chat, not an email

## When to Use LinkedIn
- As a secondary channel after email (per workflow planner rules)
- When you have no email address
- When the recipient is active on LinkedIn (recent posts)

## Do
- Reference a specific post or accomplishment
- Keep it to 2-3 sentences max for first message
- Ask a genuine question

## Don't
- Copy-paste the email message to LinkedIn
- Send connection request + pitch in the same message
- Use LinkedIn for long-form communication

## Connection Request Note
```
Hi {{first_name}} — [1 sentence context for connecting]. Would love to connect.
```
