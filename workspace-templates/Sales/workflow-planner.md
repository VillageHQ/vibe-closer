# Workflow Planner

## Outreach Sequencing Rules

1. Always reach out to the target directly
2. Fetch paths using {{FETCH_RELATIONSHIPS}}
3. When a warm connection exists, mention them in the email as social proof
4. If no reply after 3 days, send follow-up (in recipient's timezone) and send a LinkedIn message
5. Always draft a reply and follow-up on the last message
6. If the last email is from me, never follow-up more than once per channel (email/LinkedIn)
