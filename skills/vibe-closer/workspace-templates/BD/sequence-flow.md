# Sequence Flow

> Messaging tone, email rules, and LinkedIn rules → `messaging-guidelines/`
> Pipeline stages and transition rules → `pipeline-config.md` → `## Pipeline Stages`

## Execution Sequence

For each lead, follow these steps:

1. Fetch warm paths via {{FETCH_RELATIONSHIPS}}
2. Send initial outreach via the 1st path — warm intros preferred; cold email with mutual value pitch if no warm path
3. If no reply after 5 days, send follow-up (in recipient's timezone)
4. If no reply after 2 days, send a LinkedIn message
5. Always draft a reply and follow-up on the last message
6. If the last message is from me, never follow-up more than once per channel
7. After exploratory call, follow up within 24 hours with written summary and proposed next steps
8. If the lead replies and is not interested, mark as Passed in {{CRM_TRACKER}}
9. If no reply after completing all paths, set follow-up date to 6 months from now
10. If still no reply after the 6-month follow-up, set follow-up date to 1 year from now

## Frequency Rules

- Max 1 follow-up per channel before waiting 7 business days
- Proposal response wait: 7-10 business days before follow-up
- Post-negotiation: weekly check-ins during active negotiation
- Expect 4-8 weeks from first contact to signed agreement

## Key Principles

- **Lead with mutual value** — articulate what's in it for them, not just for you.
- **Propose specific collaboration ideas** — name the partnership type and first milestone. Don't ask vague "partnership" questions.
- **Respect decision-making cycles** — partnerships take longer than sales; never pressure for speed.

## Weekly Partnership Review

Every Monday, review:
- Which prospects need follow-up this week?
- Which exploratory calls are scheduled?
- Are any proposals waiting past the 10-day mark?
- New warm intro paths for stalled prospects?
