# Learn & Improve

Invoke the `vibe-closer` skill using the Skill tool, then execute this workflow:

1. Read `actions/learn.md` for the full learning process
2. Generate learnings from:
   - Communication patterns (what messaging works)
   - Workflow effectiveness (what sequences convert)
   - Profile accuracy (are ICPs correct)
3. Ask clarifying questions before persisting changes
4. Update relevant workspace files (messaging templates, ICPs, workflow planner)
5. Generate report in `progress/learnings/`
6. Update `{{LAST_LEARNING_DATE}}` in config.md
