# Learn & Improve

Invoke the `vibe-closer` skill using the Skill tool, then execute this workflow:

## Phase 1: Load Learning Process
Read `actions/learn.md` for the full learning process.

## Phase 2: Generate Learnings
Generate learnings from:
- Communication patterns (what messaging works)
- Workflow effectiveness (what sequences convert)
- Profile accuracy (are ICPs correct)

## Phase 3: Clarify & Confirm
Ask clarifying questions before persisting changes.

## Phase 4: Update Workspace Files
Update relevant workspace files (messaging templates, ICPs, workflow planner).

## Phase 5: Report & Record
Generate report in `progress/learnings/` and update `{{LAST_LEARNING_DATE}}` in config.md.
