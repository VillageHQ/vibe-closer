# v1.31.1 — Gmail-like HTML Email Editor & Activity Type Fix

## Changes

- **Modified**: `activity-viewer.html` — Squire (Fastmail's rich text editor) replaces basic contenteditable for email editing; Gmail-like toolbar with bold, italic, underline, strikethrough, lists, links, quote, undo/redo; white editor background; body history renders HTML
- **Modified**: `activity-viewer.html` — all activity type checks now use `activityChannel()` instead of hardcoded `send_email` comparisons
- **Modified**: `pipeline-config.md` body schema — email `message` field is now HTML, added `content_type: "text/html"`

## Migration Steps

1. Copy the updated `activity-viewer.html` from `skills/vibe-closer/views/activity-viewer.html` to the workspace directory (overwrite existing)
2. In workspace `pipeline-config.md` → `## Channels` → `### email` → **Body Schema**, update the `message` field description and add `content_type`:
   - Change `"message": "string"` to `"message": "string (HTML — use <p> for paragraphs, <a> for links)", "content_type": "text/html"`
3. Normalize any existing activities with bare channel names as `activity_type`:
   ```sql
   UPDATE {{ACTIVITIES_TABLE}} SET activity_type = 'send_' || activity_type WHERE activity_type IN ('email', 'linkedin', 'twitter') AND NOT activity_type LIKE 'send_%';
   ```
