# v1.31.0 — Pipeline Viewer with CRM Leads

## Changes

- **New file**: `.config/pipeline-view-config.js` — externalized viewer config (Supabase + CRM credentials), generated at setup time
- **Modified**: `activity-viewer.html` — now static (no token replacement), loads config from `.config/pipeline-view-config.js`
- **New layout**: vertical sidebar with "View Leads" and "View Activities" navigation
- **New feature**: View All Leads from CRM with inline editing (stage dropdown, follow-up date picker)
- **New field**: `{{WORKSPACE_VERSION}}` added to `pipeline-config.md` → `## Meta` section
- **New command**: `/update-skill` — user-triggered workspace updates with migration support
- **New action**: `setup-workspace-artifacts` — shared artifact generation (used by `/onboard` and `/update-skill`)
- **Removed**: `sync-with-skill` action (replaced by `/update-skill` + `setup-workspace-artifacts`)

## Migration Steps

1. Run `actions/setup-workspace-artifacts.md` to generate `.config/pipeline-view-config.js` and copy the new static `activity-viewer.html`
2. Add `- **{{WORKSPACE_VERSION}}**: {current_plugin_version}` to `## Meta` in `pipeline-config.md` if the field is not already present
